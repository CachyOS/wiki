CachyOS lze nainstalovat pomocí dvou typů instalátorů: grafického (GUI) instalátoru a instalátoru příkazové řádky (CLI).

## Kontrola

:::caution[WARNING]
Vždy udělejte extra krok a ověřte integritu ISO souboru, abyste předešli nežádoucím problémům při instalaci nebo při vytváření bootovatelných médií.
:::

Zde je jak to můžete udělat:

Pokud jsi na Windowsu:

<Steps>

1. Pokud jste stáhli ISO soubor pomocí SourceForge, vedle každého souboru je vykřičník obsahující jak SHA1, tak SHA256 kontrolní součet.

2. Otevřete CMD nebo PowerShell jako správce a navigujte do cesty, kde je uložený ISO soubor.

3. Zadejte následující příkaz: `certUtil -hashfile full_iso_name.iso SHA256`

   e.g:
   ```powershell
   certUtil -hashfile cachyos-kde-linux-230813.iso SHA256
   ```

4. Porovnejte hash získaný pomocí certUtil s informacemi o souboru na SourceForge. Pokud se shodují, jste připraveni pokračovat.

</Steps>

Kontrola z jakékoliv Linux distribuce:

<Steps>

1. Otevřete terminál a přejděte do cesty, kde je uložený ISO soubor.

2. Zadejte následující příkaz: `sha256sum full_iso_name.iso`

   Například:
   ```sh
   sha256sum cachyos-kde-linux-230319.iso
   ```

3. Porovnejte, zda se SHA256 hash získaný pomocí příkazu `sha256sum` shoduje s SHA256 hashem uvedeným na SourceForge.

</Steps>

## Ověřte autentičnost ISO souboru

Pro ověření autentičnosti ISO souboru a ujištění se, že skutečně vyšel od oficiálního vývojového týmu Athena OS:

<Steps>

1. Importujte GPG klíč pro ověření autentičnosti:

   ```shell
   gpg --keyserver hkps://keys.openpgp.org --recv-key F3B607488DB35A47
   ```

2. Stáhněte ISO soubor a jeho soubor s podpisem `.sig` a spusťte následující příkaz (nahraďte `full_iso_name.iso` skutečným názvem souboru ISO):
   ```shell
   gpg --verify full_iso_name.iso.sig full_iso_name.iso
   ```

   Pokud získáte výstup **Good signature**, ISO soubor je pravý:
   ```
   gpg: Signature made Mo 01 Apr 2024 14:13:30 CEST
   gpg:                using RSA key 882DCFE48E2051D48E2562ABF3B607488DB35A47
   gpg: Good signature from "CachyOS <admin@cachyos.org>" [unknown]
   gpg: WARNING: This key is not certified with a trusted signature!
   gpg:          There is no indication that the signature belongs to the owner.
   Primary key fingerprint: 882D CFE4 8E20 51D4 8E25  62AB F3B6 0748 8DB3 5A47
   ```

</Steps>

:::caution
Pokud výstup nevrátí řetězec **Good signature** nebo se ID klíče neshoduje, nepoužívejte ISO obraz a ověřte, zda jste soubor stáhli z legitimního zdroje Athena OS. To může naznačovat, že váš obraz byl pozměněn.
:::
