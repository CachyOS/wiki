## 1\. Aktulizuj system:

### 1\. aktulizuj system s Octopi:

Octopi je grafický správce balíčků pro distribuce založené na Archu, který usnadňuje správu balíčků a aktualizací.
Postup pro aktualizaci systému pomocí Octopi:

<Steps>

1. Spusťte **Octopi** z nabídky aplikací.
2. Klikněte na tlačítko **Systémová aktualizace** v hlavním okně Octopi.
3. Octopi zkontroluje dostupné aktualizace a zobrazí je.
4. Klikněte na tlačítko **Aplikovat**, abyste pokračovali v aktualizaci.
5. Octopi stáhne a nainstaluje aktualizace.
6. Doporučuje se restartovat počítač po velké aktualizaci, zejména pokud došlo ke změně jádra.

</Steps>

### 2\. Aktualizace systému pomocí Pacman:

<Steps>

1. otevřete terminál (nebo zmačkněte `ctrl + alt + t` - `mod + return` v Cachy WMs)
2. spusťte následující příkaz:

   ```sh
   sudo pacman -Syu
   ```

</Steps>

To je ono! nyní je systém aktulizován pro používání

## 2\. Zapněte Firewall:

Následujte tyto kroky pro zapnutí firewallu

<Tabs>

<TabItem label="1">

```bash
# Instalace ufw balíčku (Uncomplicated Firewall) pomocí Pacman:
sudo pacman -S ufw
```

</TabItem>
<TabItem label="2">

```bash
# Zapnutí firewallu tímto příkazem:
sudo ufw enable
```

</TabItem>
<TabItem label="3">

Defaultně ufw povoluje veškerý příchozí i odchozí provoz, můžete přidat specifická pravidla firewallu pro blokování nebo povolení specifických spojení.

```bash
# Například:
sudo ufw allow ssh
```

</TabItem>
<TabItem label="4">

```bash
# Pro zjištění stavu firewallu použijte následující příkaz:
sudo ufw status verbose
```

</TabItem>
</Tabs>

:::note
Buďte opatrní při konfiguraci pravidel firewallu, špatně nastavená pravidla mohou vést k uzamčení z vašeho systému.
:::

## 3\. Instalace aplikací:

CachyOS obsahuje mnoho užitečných aplikací, ale můžete chtít instalovat další podle svého pracovního postupu.
Zde jsou některé populární aplikace:

- GIMP (editor obrázků)
- VLC (mediální přehrávač)
- Stacer (monitor systému)
- Skype, Telegram, Discord, Signal (messenger aplikace)
- Steam (pro hraní her)
- Spotify (hudba)
- MailSpring (emailový klient)
- Super Productivity (manažer seznamu úkolů a Pomodoro časovač)
- Visual Studio Code (editor kódu)
- Blender (3D software)
- Krita (digitální malba)

Instalaci těchto aplikací provedete snadno přes příkazovou řádku, například:

```sh
paru -S vlc mailspring spotify gimp
```

Pokud se objeví chybová zpráva, zkuste jiný příkaz nebo zkontrolujte jméno aplikace v databázi.

## 4\. Zapnutí globálního menu:

Pro některé aplikace, jako je Visual Studio Code, může globální menu nefungovat správně nebo může být připojeno k rodičovské aplikaci místo panelu. Pro zapnutí podpory globálního menu spusťte následující příkaz a restartujte aplikaci:

```sh
# Pro zapnutí globálního menu spusťte příkaz
sudo pacman -S appmenu-gtk-module libdbusmenu-glib
```
