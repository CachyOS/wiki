# Často kladené otázky

Zde jsou některé často kladené otázky. Doporučujeme, aby si je přečetli uživatelé používající CachyOS nebo distribuce založené na Arch Linuxu.

## FAQ k řízení softwaru

Zde jsou všechny otázky související s řízením softwaru ve vašem systému.

### Jak opravit chybu "unable to lock database" v CachyOS?

```sh
sudo rm /var/lib/pacman/db.lck
```

:::tip
Toto lze také provést kliknutím na tlačítko v grafickém rozhraní CachyOS Hello.
:::

### Jak opravit možné problémy s keyringy v CachyOS?

```sh
sudo rm -rf /etc/pacman.d/gnupg/
sudo pacman-key --init
sudo pacman-key --populate

sudo pacman-key --recv-keys F3B607488DB35A47 --keyserver keyserver.ubuntu.com
sudo pacman-key --lsign-key F3B607488DB35A47

sudo rm -R /var/lib/pacman/sync
```

### Jak odstranit tyto otravné varování při aktualizaci systému? Je bezpečné aktualizovat systém, když se toto stane?

"warning: X package: local(1.0.1-1.1) is newer than Y repo (1.0.1-1)"

<MultipleImageComponent
  images={[import('~/assets/images/update-warnings.png')]}

Ano, aktualizace systému je bezpečná.

Chcete-li odstranit tyto chyby, proveďte aktualizaci s '-Syuu':

```sh
sudo pacman -Syuu
```
