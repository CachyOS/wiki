## 1\. Aktualizujte svoj systém:

### 1\. Aktualizácia systému pomocou Octopi:

Octopi je grafický správca balíčkov pre distribúcie založené na Archu, ktorý poskytuje pohodlný spôsob správy balíčkov a aktualizácií.
Na aktualizáciu systému pomocou Octopi postupujte podľa týchto krokov:

<Steps>

1. Spustite **Octopi** z aplikačného menu.
2. V hlavnom okne kliknite na tlačidlo **System Upgrade**.
3. Octopi skontroluje dostupné aktualizácie a predloží ich vám.
4. Na pokračovanie v aktualizácii kliknite na tlačidlo **Apply**.
5. Octopi stiahne a nainštaluje aktualizácie.
6. Odporúča sa reštartovať počítač po veľkej aktualizácii (najmä ak sa zmení verzia jadra).

</Steps>

### 2\. Aktualizácia systému pomocou Pacman:

<Steps>

1. Otvorte terminálový emulátor (alebo stlačte `ctrl + alt + t` - `mod + return` v Cachyho WM).
2. Spustite nasledujúci príkaz na aktualizáciu systému:

   ```sh
   sudo pacman -Syu
   ```

</Steps>

A je to! Teraz je váš systém aktuálny a pripravený na použitie.

## 2\. Povoliť ochranu firewallu:

Na povolenie ochrany firewallu postupujte podľa týchto krokov:

<Tabs>

<TabItem label="1">

```bash
# Nainštalujte balík ufw (Uncomplicated Firewall) pomocou Pacman:
sudo pacman -S ufw
```

</TabItem>
<TabItem label="2">

```bash
# Povoľte firewall pomocou tohto príkazu:
sudo ufw enable
```

</TabItem>
<TabItem label="3">

Predvolene ufw povolí všetku prichádzajúcu a odchádzajúcu komunikáciu. Môžete pridať konkrétne pravidlá do firewallu na blokovanie alebo povolenie konkrétnych pripojení.

```bash
# Napríklad:
sudo ufw allow ssh
```

</TabItem>
<TabItem label="4">

```bash
# Na kontrolu stavu firewallu použite nasledujúci príkaz:
sudo ufw status verbose
```

</TabItem>
</Tabs>

:::note
Buďte opatrní pri konfigurácii pravidiel firewallu, pretože nesprávne nakonfigurované pravidlá vás môžu zablokovať z vlastného systému.
:::

## 3\. Inštalácia aplikácií:

CachyOS prichádza predinštalovaný s mnohými užitočnými aplikáciami, ale možno budete chcieť nainštalovať ďalšie, aby zodpovedali vášmu pracovnému toku.
Tu sú niektoré populárne aplikácie, ktoré by ste mohli zvážiť nainštalovať:

*   GIMP (editor obrázkov)
*   VLC (prehrávač médií)
*   Stacer (systémový monitor)
*   Skype, Telegram, Discord, Signal (messenger aplikácie)
*   Steam (pre hranie)
*   Spotify (hudba)
*   MailSpring (e-mailový klient)
*   Super Productivity (manažér úloh a Pomodoro timer)
*   Visual Studio Code (editor kódu)
*   Blender (3D softvér)
*   Krita (digitálne maľovanie)

Tieto aplikácie môžete jednoducho nainštalovať pomocou príkazového riadku. Napríklad:

```sh
paru -S vlc mailspring spotify gimp
```

Ak dostanete chybovú správu, skúste použiť iný príkaz alebo skontrolujte názov aplikácie v databáze.

## 4\. Povoliť globálne menu:
Pre niektoré aplikácie ako Visual Studio Code, globálne menu nemusí fungovať alebo môže byť pripojené k nadradenej aplikácii namiesto panela.

```sh
# Na povolenie podpory globálneho menu spustite tento príkaz a reštartujte aplikáciu.
sudo pacman -S appmenu-gtk-module libdbusmenu-glib
```

## 5\. Nastavenie Bluetooth slúchadiel:
Ak chcete automaticky pripojiť svoje slúchadlá, postupujte podľa krokov v príručke Arch wiki: [https://wiki.archlinux.org/title/bluetooth\_headset#Headset\_via\_Bluez5/PulseAudio](https://wiki.archlinux.org/title/bluetooth_headset#Headset_via_Bluez5/PulseAudio). Ak Pulseaudio nefunguje, môže byť potrebné manuálne znovu pripojiť slúchadlá pri každom reštarte počítača.
